### Technologies Used
The application is built using the following technologies:

- React Js
- .Net 6 web API
- Nginx
- Docker
- Linux
- SQL Server

## Steps to start the app

### Start the react app
Go to clinicnanagement-app folder
```
cd clinicnanagement-app
```
Install the node modules
```
npm install
```
Start the react app
```
npm start
```

### Start the backend server
Open a new terminal
Go to clinicnanagement folder
```
cd clinicnanagement
```
Install the .net 
```
dotnet build
```
Start the server
```
dotnet run
```
